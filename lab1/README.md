# Лабораторная работа 1: Перемножение квадратных матриц

## Цель работы

Написать программу на языке C/C++ для перемножения двух квадратных матриц, с автоматизированной верификацией результатов вычислений с помощью сторонних библиотек (Python).

## Задание

- **Реализовать** программу на языке C/C++ для перемножения **двух квадратных матриц**.
- **Исходные данные**: файл(ы), содержащие значения исходных матриц.
- **Выходные данные**: файл со значениями результирующей матрицы, время выполнения, объем задачи (количество операций).
- **Обязательна автоматизированная верификация** результатов вычислений с помощью сторонних библиотек (например, Python `numpy`).

## Структура проекта

- `main.cpp`: Реализация алгоритма умножения матриц на C++.
- `verif.py`: Python-скрипт для автоматизированной верификации:
  - Генерация случайных матриц заданных размеров.
  - Запуск исполняемого файла `main.exe`.
  - Сравнение результата с эталонным вычислением (с использованием `numpy`).
  - Замер времени выполнения.
  - Сохранение результатов в `results.csv`.
  - Построение графика зависимости времени от размера матрицы (`plot.png`).
- `results.csv`: Результаты бенчмарка (размер матрицы, время выполнения).
- `plot.png`: График производительности.
- `M1.txt`, `M2.txt`: Примеры входных файлов (не обязательны, если `verif.py` генерирует свои).

## Реализация на C++

Программа `main.cpp`:
- Читает размер матрицы `n` и элементы двух квадратных матриц `A` и `B` из файлов, указанных в аргументах командной строки.
- Выполняет умножение матриц `C = A * B` по классическому алгоритму (три вложенных цикла).
- Записывает результирующую матрицу `C` в выходной файл.
- Выводит в консоль размер матрицы, количество арифметических операций (`n^3`) и время выполнения.

## Верификация на Python

Скрипт `verif.py`:
- Использует `numpy` для генерации случайных матриц и эталонного вычисления их произведения.
- Запускает исполняемый файл `main.exe` через `subprocess`.
- Сравнивает результат, вычисленный `main.exe`, с результатом `numpy` (с учётом погрешности вычислений с плавающей точкой).
- Автоматически запускает серию тестов на разных размерах матриц.
- Сохраняет замеры времени в `results.csv` и строит график в `plot.png`.
