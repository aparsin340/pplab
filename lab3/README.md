## Цель работы

Написать программу на языке C++ для параллельного перемножения двух квадратных матриц с использованием библиотеки MPI, с автоматизированной верификацией результатов вычислений и оценкой производительности.

## Задание

- **Реализовать** программу на языке C++ для **параллельного перемножения** двух квадратных матриц.
- **Использовать** библиотеку **MPI (Message Passing Interface)** для распараллеливания вычислений между процессами.
- **Исходные данные**: файл(ы), содержащие значения исходных матриц.
- **Выходные данные**: файл со значениями результирующей матрицы, время выполнения, объем задачи (количество операций), производительность (GFLOPS).
- **Обязательна автоматизированная верификация** результатов вычислений с помощью сторонних библиотек (например, Python `numpy`).
- **Сравнить** производительность при использовании разного количества процессов.
- **Построить** графики зависимости времени выполнения, ускорения (speedup) и эффективности (efficiency) от размера матрицы и количества процессов.

## Реализация на C++

Программа `lab3/main.cpp`:
- Читает размер матрицы `n` и элементы двух квадратных матриц `A` и `B` из файлов, указанных в аргументах командной строки.
- Выполняет умножение матриц `C = A * B` с использованием **параллельного алгоритма**:
  - Матрица `A` разбивается по строкам между процессами.
  - Матрица `B` рассылается всем процессам через `MPI_Bcast`.
  - Строка `A` умножается на всю матрицу `B` локально на каждом процессе.
  - Результат собирается на процессе 0 через `MPI_Gatherv`.
- Записывает результирующую матрицу `C` в выходной файл.
- Выводит в консоль размер матрицы, количество арифметических операций (`n^3`), время выполнения и производительность (GFLOPS).
- Использует `MPI_Init`, `MPI_Comm_rank`, `MPI_Comm_size`, `MPI_Bcast`, `MPI_Scatterv`, `MPI_Gatherv`, `MPI_Barrier`, `MPI_Wtime`, `MPI_Finalize`.

## Верификация и бенчмаркинг на Python

Скрипт `lab3/verif.py`:
- Использует `numpy` для генерации случайных матриц и эталонного вычисления их произведения.
- Запускает исполняемый файл `main.exe` через `mpiexec` с разным количеством процессов.
- Сравнивает результат, вычисленный `main.exe`, с результатом `numpy` (с учётом погрешности вычислений с плавающей точкой).
- Автоматически запускает серию тестов на различных размерах матриц и количестве процессов.
- Собирает статистику (среднее время, стандартное отклонение) и сохраняет её в `results.csv`.
- Строит графики `plot_time.png`, `plot_speedup.png`, `plot_efficiency.png`.
