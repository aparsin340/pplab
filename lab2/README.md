## Цель работы

Написать программу на языке C++ для параллельного перемножения двух квадратных матриц с использованием библиотеки OpenMP, с автоматизированной верификацией результатов вычислений и оценкой производительности.

## Задание

- **Реализовать** программу на языке C++ для **параллельного перемножения** двух квадратных матриц.
- **Использовать** библиотеку **OpenMP** для распараллеливания вычислений.
- **Исходные данные**: файл(ы), содержащие значения исходных матриц.
- **Выходные данные**: файл со значениями результирующей матрицы, время выполнения, объем задачи (количество операций), производительность (GFLOPS).
- **Обязательна автоматизированная верификация** результатов вычислений с помощью сторонних библиотек (например, Python `numpy`).
- **Сравнить** производительность параллельной и последовательной версий.
- **Построить** графики зависимости времени выполнения и ускорения от размера матрицы и количества потоков.

## Структура проекта

- `lab2/main.cpp`: Реализация последовательного и параллельного алгоритмов умножения матриц на C++ с использованием OpenMP.
- `lab2/verif.py`: Python-скрипт для автоматизированной верификации и бенчмаркинга:
  - Генерация случайных матриц заданных размеров.
  - Запуск исполняемого файла `main.exe` с разным количеством потоков.
  - Сравнение результатов с эталонным вычислением (с использованием `numpy`).
  - Замер времени выполнения для различных конфигураций (размер, потоки).
  - Сохранение результатов в `results.csv`.
  - Построение графиков зависимости времени от размера/количества потоков (`plot_time.png`) и ускорения (`plot_speed.png`).
- [`lab2/results.csv`](lab2/results.csv): Результаты бенчмарка (размер матрицы, количество потоков, среднее время выполнения, стандартное отклонение).
- [`lab2/plot_time.png`](lab2/plot_time.png): График зависимости времени выполнения от размера матрицы и количества потоков.
- [`lab2/plot_speed.png`](lab2/plot_speed.png): График ускорения (Speedup) при использовании нескольких потоков.
- `lab2/M1.txt`, `lab2/M2.txt`: Примеры входных файлов (не обязательны, если `verif.py` генерирует свои).

## Реализация на C++

Программа `lab2/main.cpp`:
- Читает размер матрицы `n` и элементы двух квадратных матриц `A` и `B` из файлов, указанных в аргументах командной строки.
- Выполняет умножение матриц `C = A * B` с возможностью выбора между **последовательной** и **параллельной** версией.
- **Параллельная версия** использует директиву `#pragma omp parallel for` для распараллеливания внешнего цикла по строкам результирующей матрицы.
- Записывает результирующую матрицу `C` в выходной файл.
- Выводит в консоль размер матрицы, количество арифметических операций (`n^3`), время выполнения и производительность (GFLOPS).
- Принимает дополнительные аргументы командной строки для указания количества потоков или запуска последовательной версии.

## Верификация и бенчмаркинг на Python

Скрипт `lab2/verif.py`:
- Использует `numpy` для генерации случайных матриц и эталонного вычисления их произведения.
- Запускает исполняемый файл `main.exe` через `subprocess` с разными параметрами (размер, потоки).
- Сравнивает результат, вычисленный `main.exe`, с результатом `numpy` (с учётом погрешности вычислений с плавающей точкой).
- Автоматически запускает серию тестов на различных размерах матриц и количестве потоков.
- Собирает статистику (среднее время, стандартное отклонение) и сохраняет её в `results.csv`.
- Строит графики `plot_time.png` и `plot_speed.png`.
